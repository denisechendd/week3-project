# 第三週作業｜開始熟練 React

本專案為六角學院 React 課程第三週作業，主要延續前一週的 API 串接練習，進一步熟悉 React 中的狀態管理、表單操作與 Bootstrap Modal 的使用，並完成後台產品的新增、編輯與刪除功能。

---

## 🎯 作業目標

完成此作業可練習與理解以下技能：

- React 狀態管理（useState）
- useEffect 搭配非同步流程的實作
- RESTful API CRUD 操作
- 表單資料處理與送出
- Bootstrap Modal 的顯示與關閉控制
- API 操作後的畫面同步更新

---

## 📌 功能說明

- 管理者登入驗證
- 取得並顯示產品列表
- 新增產品資料
- 編輯既有產品資料
- 刪除產品
- 透過 Modal 進行新增 / 編輯 / 刪除操作
- 操作完成後即時重新取得產品資料
- 基本錯誤處理提示
